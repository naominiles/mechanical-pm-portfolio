# Mechanical PM - Portfolio Website

Professional portfolio website for Zack Lancaster, Mechanical Project Manager based in Las Vegas, NV.

## Overview

This is a clean, professional portfolio website showcasing mechanical project management experience across healthcare, commercial, and large-scale entertainment construction projects.

## Features

- **Homepage** - Introduction and selected projects
- **Projects** - Full portfolio with filtering
- **Project Details** - In-depth case studies
- **Insights** - Blog and writing
- **Applied A.I.** - Documentation of AI tools in project management
- **About** - Professional background and experience
- **Contact** - Get in touch form

## Design

- Typography: DM Serif Display & DM Sans
- Color Palette: Warm neutrals with bronze accents
- Style: Minimalist, architectural, professional

## Tech Stack

- Pure HTML/CSS
- No frameworks or build process
- Responsive design
- Self-contained pages

## Local Development

Simply open any `.html` file in a web browser to view the site locally.

## Structure

```
├── index.html           # Homepage
├── projects.html        # Projects listing
├── project-detail.html  # Project case study
├── insights.html        # Blog/insights
├── applied-ai.html      # AI tools page
├── about.html           # About page
├── contact.html         # Contact form
└── zackary-lancaster.png # Profile photo
```
